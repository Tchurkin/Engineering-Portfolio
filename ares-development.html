Ares Development

Flight-by-flight evolution of the Ares TVC stack. Hardware, control loops, and failure analysis in one place.

Inspired by the project-story style from AeroFusion, this page tracks the iterative build history from bench checkout to controlled ascent and post-burnout instability analysis.

Milestone Timeline Subsystem Changes Flight Matrix Evidence Links

Milestone Timeline

Phase 01

Static Bench Validation

Validated sensor fusion stability, servo authority, and Teensy loop timing under sustained test runs.

Phase 02

First Controlled Flights

Closed-loop PD control delivered visible attitude stabilization during powered ascent.

Phase 03

Gimbal and Wiring Iteration

Reduced mechanical play and electrical noise to improve command fidelity near peak dynamic pressure.

Phase 04

Burnout Transition Analysis

Characterized phase lag and stability loss at thrust cutoff to define next-controller requirements.

Subsystem Changes

Guidance and Control

PD baseline implemented with gain sweeps and saturation limits for safer flight envelope exploration.

Sensing and Estimation

BNO055 plus DPS310 data streams synchronized in firmware for cleaner attitude and pressure trends.

TVC Mechanics

Gimbal geometry and linkage stiffness improved to reduce backlash-driven oscillations.

Flight Firmware

State machine cleanup for launch detect, powered stabilization, and post-burnout handling logic.
