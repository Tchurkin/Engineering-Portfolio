PID Demo Box

Interactive control-system demo. Real-time PID response and tuning shown through physical hardware.

Category: Controls / Embedded Systems / Education

Timeline: July 2025

Status: Demonstrated at Open Sauce; iterating on hardware and software polish.

Problem: PID behavior can be hard to understand without seeing how gain changes affect a real system.

My role: Designed and built the demo box, implemented embedded control logic, and tuned the loop for stable and readable live demonstrations.

Goal: Make control theory intuitive by showing overshoot, damping, and settling-time changes in real time as gains are adjusted.
